# Ajinomoto-conductor

## Project Overview
Ajinomoto-conductor is an automated data analysis and reporting workspace. It utilizes custom Python scripts and specialized Gemini CLI skills to extract, analyze, and format game performance data (specifically for "Ajipanda's Kitchen"). It correlates raw data from Excel spreadsheets with key findings from PowerPoint reports to generate structured, easy-to-understand JSON summaries and dynamically update Excel charts.

## Key Files and Directories
- **`conductor/plan.md`**: The central implementation plan outlining the objectives and steps for creating specialized Gemini skills (like the `excel-graph-updater`).
- **Data Sources**: 
  - `Aji_game copy.xlsx`: The primary Excel file containing raw game performance data across multiple sheets (User Funnel, User Engagement, Game Performance, etc.).
  - `Merkle Thailand -Ajipanda's Kitchen report- 260331 copy.pptx`: The PowerPoint presentation containing the key findings and report summaries.
- **Python Scripts**:
  - `inspect_excel.py`: A utility script using `openpyxl` to inspect the structure of the Excel sheets and chart references.
  - `test_regex.py` & `test_regex_chart.py`: Scripts containing logic for parsing and modifying chart data references (e.g., XML elements) within the Excel structure.
- **`.gemini/skills/`**: The directory containing custom Gemini CLI skills developed for this project.
  - `excel-graph-updater`: A skill designed to automatically update Excel graphs to point to data for a specific month.
  - `excel-data-analyzer`: A skill that extracts data from both the Excel and PPTX files, correlates the metrics with the key findings, and outputs the result as a structured JSON file.
- **`analysis_output.json`**: The resulting JSON output generated by the `excel-data-analyzer` skill, representing the final structured analysis.

## Usage and Development
This workspace is designed to be interacted with via the Gemini CLI, leveraging the custom skills. 

**The End-to-End Generation Workflow:**
1.  **Sync Excel:** Run `excel-graph-updater` (or let the generator handle it) to slice the Excel charts for the target month.
2.  **Generate Insights (JSON):** Invoke the `excel-data-analyzer` skill to correlate the new Excel data against the template and generate tailored insights. This outputs a JSON file (e.g., `analysis_output_[Month].json`).
3.  **Generate PPTX:** Invoke the `aji-report-generator` skill (e.g., "Give me a report in [Month]"). This script will parse the newly created JSON insights and surgically construct the PowerPoint presentation, injecting both the hard metrics and the custom `key_finding` paragraphs.

- The Python scripts in the root directory serve as testing and inspection utilities for the core logic used within the skills.
